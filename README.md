# Dashboard de Videojuegos (Power BI)

**Dashboard interactivo para analizar ventas y tendencias en la industria de videojuegos.**

---

## Resumen del Proyecto

Este proyecto presenta un dashboard desarrollado en Power BI que permite visualizar métricas de ventas de videojuegos, detectar patrones por plataforma, género y región, y entender tendencias generales de mercado.  
El dashboard facilita el análisis rápido de datos de videojuegos para apoyar decisiones basadas en información.

---

## Impacto en el Negocio / Resultados Clave

- Identificación de los **videojuegos con mayor número de ventas** por región y género.  
- Visualización de tendencias en datos clave para decisiones de inventario y marketing.  
- Permite detectar **patrones de consumo** por plataforma o período de tiempo.  
- Dashboard interactivo que puede ser usado por equipos no técnicos para explorar datos complejos.

---

## Tecnologías Utilizadas

- Power BI Desktop  
- Microsoft Excel o CSV como fuente de datos  
- Modelado y transformación de datos  
- Visualizaciones interactivas con DAX y filtros

---

## Funcionalidades Clave

- **Resumen de Ventas Globales:** Total de ventas y unidades vendidas por juego.  
- **Análisis por Plataforma:** Comparación entre diferentes plataformas de juego.  
- **Filtros Dinámicos:** Permite segmentar datos por año, género, región o plataforma.  
- **Visualizaciones Interactivas:** Gráficos y tablas para explorar tendencias y patrones.

## Cómo Usar

1. Descarga el archivo `.pbix` incluido en el repositorio.  
2. Abre el archivo con **Power BI Desktop (versión 2022 o superior)**.  
3. Conecta los datos o utiliza los archivos de ejemplo si los proporcionas.  
4. Explora los filtros y visualizaciones interactivas para obtener insights.
